# 轻量级博客平台
基于 Spring Boot + Vue 的前后端分离式博客系统

## 项目概述
本系统为前后端分离式轻量级博客平台，采用「用户端 + 管理端」双端架构，覆盖 UGC 内容管理、用户认证、个性化推荐、后台运维全场景。后端基于 Spring Boot 3 提供 RESTful API，前端基于 Vue 3 实现交互界面，整合 DFA 敏感词过滤、协同过滤推荐算法，构建功能完备、安全可控的 Web 应用。

### 核心目标
1. 技术整合：落地 Spring Boot、MyBatis-Plus、Vue 等技术，掌握前后端分离接口联调
2. 算法落地：实现 DFA 敏感词过滤、协同过滤个性化推荐
3. 业务闭环：覆盖用户认证、内容管理、权限控制、数据可视化等核心场景
4. 工程化：规范分层开发，提升系统可扩展性与可维护性

## 技术栈
### 后端技术
- 核心框架：Spring Boot 3
- 数据持久化：MyBatis-Plus
- 认证授权：JWT + BCrypt 加密
- 算法：DFA（敏感词过滤）、Mahout 协同过滤（个性化推荐）
- 工具类：Lombok、Hutool、Apache POI（Excel 导入导出）、JavaMail（邮箱验证码）
- 日志/异常：SLF4J 日志、全局异常处理

### 前端技术
- 框架：Vue 3 + Element Plus
- 路由/状态：Vue Router、Pinia
- 网络请求：Axios
- 可视化：ECharts
- 构建工具：Vite

### 运行环境
- 操作系统：Windows 10/11 64位
- JDK：17
- 数据库：MySQL 8.0
- 项目管理：Maven 3.9.6
- 开发工具：IntelliJ IDEA、VS Code

## 核心功能模块
### 1. 管理端（管理员专属）
- 登录管理：JWT 令牌生成与校验，权限路由跳转
- 数据可视化：基于 ECharts 展示用户注册量、帖子发布量等核心数据
- 用户管理：条件分页查询、Excel 批量导入导出、状态管控
- 内容管理：帖子/分类/评论/公告/敏感词的增删改查
- 敏感词管理：DFA 字典树初始化，支持敏感词动态刷新

### 2. 用户端（普通用户）
- 认证功能：账号密码登录、邮箱注册、验证码找回密码
- 内容消费：首页个性化推荐（协同过滤）、帖子/公告查看、评论互动
- 内容生产：帖子发布、评论回复（敏感词自动过滤）
- 个人中心：个人信息管理、创作内容查看

## 核心技术实现
### 1. 敏感词过滤（DFA 算法）
- 项目启动时初始化敏感词字典树，支持实时刷新
- 核心逻辑：遍历字符匹配字典树，识别敏感词并处理
- 应用场景：帖子发布、评论回复的内容合规校验

### 2. 个性化推荐（协同过滤）
- 基于用户行为数据构建偏好模型，采用欧氏距离计算用户相似度
- 降级策略：无数据/未登录用户返回随机推荐内容，保障可用性

### 3. 权限与安全
- JWT 生成令牌，拦截器验证权限，区分管理员/普通用户路由
- 密码采用 BCrypt 加密，验证码有效期管控，全局异常处理

### 4. 数据操作
- MyBatis-Plus 实现分页查询、条件筛选
- Apache POI 完成用户数据的 Excel 导入导出

| blog_user    | 用户信息存储             | user_id、username、password、user_type |

| comment      | 评论内容存储             | id、post_id、user_id、content、parent_id |
